What is Forge Documentation Press?

Forge Documentation Press is a pre-packaged DocBook XML tool chain. It uses comes with a collection of Java tools to process DocBook XML into HTML, PDF, and other formats. All of the tools come set up and ready to use.


What components are included?

Forge Documentation Press includes the following:

  • DocBook XSL-NS style sheets
  • Saxon 6.5.5
  • Apache FOP 0.95
  • Apache Xalan Java 2.7.1
  • Apache Commons Resolver 1.2
  • a basic customization layer


Do I need any other tools?

Yes. Forge Documentation Press uses Apache Ant (1.6.5 or later) to drive the prublication process.


What are some good resources for learning how to customize my output?
