Source

Get the Forge Documentation Press source code

The Forge Documentation Press project stores its source code in a Subversion repository at:

http://fusesource.com/forge/svn/forgedp/trunk

We use the trunk for the next main release; then we use a branch for any bug fixes on the previous major release. You can look at a specific branch here:

http://fusesource.com/forge/svn/forgedp/branches

Checking Out

Check out the Forge Documentation Press source using the following command:

svn co http://fusesource.com/forge/svn/forgedp/trunk fusedp
